Meter → Tone

Tone.Meter gets the RMS of an input signal with some averaging applied. It can also get the raw value of the signal or the value in dB. For signal processing, it's better to use Tone.Follower which will produce an audio-rate envelope follower instead of needing to poll the Meter to get the output.

Meter was inspired by Chris Wilsons Volume Meter.

new Tone.Meter ([channels][, smoothing][, clipMemory])
channels = 1 {number}

number of channels being metered

smoothing = 0.8 {number}

amount of smoothing applied to the volume

clipMemory = 0.5 {number}

number in seconds that a "clip" should be remembered

code
var meter = new Tone.Meter();
var mic = new Tone.Microphone().start();
//connect mic to the meter
mic.connect(meter);
//use getLevel or getDb 
//to access meter level
meter.getLevel();
default values
smoothing:0.8
bufferSize:1024
clipMemory:0.5
clipLevel:0.9
channels:1
members
.smoothing {NormalRange}

The amount which the decays of the meter are smoothed. Small values will follow the contours of the incoming envelope more closely than large values.

.clipMemory {Time}

The amount of time a clip is remember for.

.clipLevel {Number}

The value above which the signal is considered clipped.
