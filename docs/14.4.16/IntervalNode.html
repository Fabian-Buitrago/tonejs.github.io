IntervalNode

Represents a node in the binary search tree, with the addition of a "high" value which keeps track of the highest value of its children. References: https://brooknovak.wordpress.com/2013/12/07/augmented-interval-tree-in-c/ http://www.mif.vu.lt/~valdas/ALGORITMAI/LITERATURA/Cormen/Cormen.pdf
